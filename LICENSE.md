**Fix:** Place Lua scripts in the `YimMenu/scripts/` folder within your GTA V directory (or the folder specified in `YimMenu/settings.json`). Scripts must have a `.lua` extension and valid syntax.

### ❌ Auto-update fails or loops
**Fix:** Check your internet connection and firewall settings. The updater calls the GitHub API — ensure outbound connections to `api.github.com` are not blocked by your firewall or router.

### ❌ Settings reset after every session
**Fix:** Run your injector and GTA V as Administrator so YimMenu has write permissions to save `settings.json` between sessions.
